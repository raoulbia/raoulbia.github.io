Linux

On this page I keep snippets of code I have found useful when working in a Linux environment. Areas covered:

  • File Handling
  • File System and Environment
  • Software
  • Misc.

File Handling

  • To unzip a compressed file: $ tar -zxvf archive_name.tar.gz

  • To print out duplicate lines from file: $ uniq -d file.txt

  • To search for file

    • to search for a file named e.g. rdflib, start at root dir: $ sudo find ~/ -name rdflib

    • to search for a file with extension .cls:

      $ sudo su
$ find / -name "*.cls"

  • To remove file: $ rm fileNme

  • To remove directory: $ rm -r dirName

  • To check nature of file: $ file -b buld_kg.sh

    output: Bourne-Again shell script, ASCII text executable

  • To open file: $ less fileName, then to close and exit: $ q + enter

  • To display file content:

    $ cat <filename>
$ cat -vet <filename>  # to see hidden chars

  • To make a shortcut: $ ln -s /opt/robomongo/bin/robomongo ~/robomongo_lnk

File System and Environment

  • To kill process if file busy

    $ lsof |grep -i path-to-file/filename
$ kill -9 <pid>

  • To list installed packages: $ pip freeze

  • About the double dash --

    Almost all short options have a corresponding long option; rm -f is equivalent to rm --force.

    Many long options, though, have no corresponding short option.

  • To check Ubuntu version: $ lsb_release -a

  • To list hidden files

    $ ls -ld .?*
$ ls -lA

  • To list files installed by some package: $ dpkg -L package-name

  • To find path to a binary (exec) file

    which mongod would give you the path to the mongod binary.

  • To check if OS is 32 or 64 bit: $ uname -m

Misc.

  • About the usr/bin directory: see this SO post

  • apt-get vs. pip install vs. conda install:

    • apt-get installs a (python) package/module system-wide.
    • pip installs module in venv only.
    • If using pip install after activating the venv, module will be installed into that venv only.
    • pip installs a package/module within any environment.
    • conda installs module/package within conda environments.
