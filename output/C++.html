Rcpp, C++ and Visual Studio Code

  • Including the header Rcpp.h only works if the code is executed / compiled from RStudio
  • Compiling a pure cpp file without Rcpp export does NOT require an Rcpp header include. If you do yo will get an error: Rcpp.h: No such file or directory.
