SSH Keys

Fri 01 December 2017

Checking for existing keys

ls -al ~/.ssh

Generating a new key

ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

or

ssh-keygen -t rsa -C "some name" -b 4096

Adding a key

eval $(ssh-agent -s)
ssh-add ~/.ssh/id_rsa
