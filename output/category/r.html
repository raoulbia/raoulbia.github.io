Sliding Window

How sliding window works:

  • select subset of x of size 3 using its indices
  • 1st iter: x[3 - 3 + 1, 3] gives x[1, 3]
  • 2nd iter: x[4 - 3 + 1, 4] gives x[2, 4]
  • ...
window <- 3 
for(i in seq(window, n)) {
x_subset <- x[seq(i - window + 1, i)]
res[i] <- mean(x_subset)
}
